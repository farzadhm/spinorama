# اسپینوراما

<br />

اسپینوراما یک کامپوننت کاروسل بسیار قابل سفارشی‌سازی و چندمنظوره برای برنامه‌های **React** است که به طور خاص برای کار با **Material-UI** طراحی شده است. با اسپینوراما، می‌توانید به راحتی کاروسل‌های جذاب و تعاملی ایجاد کنید که محتوای شما را به شکلی بسیار بصری و زیبا نمایش می‌دهد.
